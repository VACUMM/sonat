Common options
==============

.. program:: sonat

These options must be placed before any sub-command,
and they are common to all sub-commands.

All these options, except the :option:`-h`,  :option:`--help`, :option:`--short-help`,
:option:`--long-help`, and :option:`--cfgfile` options,
automatically generated from the configuration specifications.
They are available to make it possible to alter configuration option
from command line.
